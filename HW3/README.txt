README
======

Purpose: 
	To verify that 4 dividend arguments are divisible by a divisor.

This package includes the following files.

|-- check.c [Checker is implemented here.]
|-- coord.c [Coordinator is implemented here.] 
|-- README.txt [This file]

To compile:
    make all

To cleanup project directory:
    make clean

To run:
    ./coordinator <divisor> <dividend1> <dividend2> <dividend3> <dividend4>

For example:
    ./coordinator 1 2 3 4 5
